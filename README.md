# manual-testing-portfolio
Basic test plan for Canamo+ Construction
Manual Testing Project — Canamo Plus Construction (canamoplus.com)

Project Overview
Manual test plan and bug report for a live Canadian construction company website in Montreal. This project was completed as requested by Canamo Project Manager.
</br>
What Was Tested</br>
Navigation and internal links</br>
Get a Quote form — field validation and boundary testing</br>
Footer contact form — validation testing</br>
Language toggle (FR/EN)</br>
Social media links (LinkedIn, Instagram)</br>
Responsiveness across desktop and mobile viewports</br>
XSS input boundary testing</br>
</br>
Results</br>
25 test cases executed</br>
19 passed, 5 failed, 1 not executed</br>
5 defects identified and logged in Jira</br>
</br>
Defects Found
CANAMO-001 - Quote form submits without Company Name - High</br>
CANAMO-002 - Quote form accepts invalid email format - High</br>
CANAMO-003 - Phone field accepts alphabetical characters - Medium</br>
CANAMO-004 - Footer form accepts invalid email format - High</br>
CANAMO-005 - Instagram social link leads to broken page - Medium</br>
</br>
Tools Used</br>
Jira — bug tracking and defect lifecycle management</br>
Excel — test plan and test case documentation</br>
Chrome DevTools — responsiveness and mobile testing</br>
